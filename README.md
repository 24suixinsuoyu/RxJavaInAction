此repository是本人的《RxJava2.x 实战》一书中的相关的例子。里面包括Android和Java相关的例子。

# 开发工具
Android的工程采用Android Studio3.1开发，gradle版本是4.1。

Java的工程采用Intellij IDEA社区版开发。
